# Governance Beschreibungen

Die Governances beschteiben im Kontext eines Einsatz durch ein Web-Analyse-Tool zur Netzwerküberwachung folgende Ziele:

##  eu-innovation.json
**European Innovation Governance** ist eine Web-Analyse-Governance zur wirtschaftlichen Verfügbarkeit und industriellen Resilienz. Sie integriert zwei Policies – *default-economic-policy* und *resilience-policy* – und steuert vier analytische Views: **multiPerspectiveAnalysis** (Mehrperspektivenanalyse), **domainSpecificEvaluation** (domänenspezifische Bewertung), **resilienceAssessment** (Resilienzprüfung) und **economicAvailability** (Wirtschaftlichkeitsprüfung). Diese Kombination ermöglicht eine strukturierte Netzwerküberwachung europäischer Innovationsinfrastrukturen.

## global-neutral.json

Die *Global Neutral Governance* stellt eine ausgewogene, politisch und regional unvoreingenommene Netzwerkanalyse sicher. Drei spezialisierte Views strukturieren die Auswertung: **multiPerspectiveAnalysis** aggregiert globale Netzwerkperspektiven ohne regionale Gewichtung, **domainSpecificEvaluation** bewertet Domains nach neutralen, domänenspezifischen Kriterien, **resilienceAssessment** analysiert Ausfallsicherheit und Verfügbarkeit weltweit verteilter Ressourcen. Default Economic und Resilience Policy bilden das gemeinsame Regelwerk für eine faire, unparteiische Erreichbarkeitsbewertung.
