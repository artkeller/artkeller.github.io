# Governance Beschreibungen^

Die Governances beschteiben im Kontext eines Einsatz durch ein Web-Analyse-Tool zur Netzwerküberwachung folgende Ziele:

##  eu-innovation.json

Das Web-Analyse-Tool zur Netzwerküberwachung implementiert die *European Innovation Governance* mit Fokus auf wirtschaftliche Verfügbarkeit und industrielle Resilienz. Vier spezialisierte Views strukturieren die Analyse: **multiPerspectiveAnalysis** liefert übergreifende Netzwerksichten, **domainSpecificEvaluation** bewertet domänenspezifische Metriken, **resilienceAssessment** prüft Ausfallsicherheit und Redundanz, **economicAvailability** misst ökonomische Erreichbarkeit kritischer Ressourcen. Zwei Policy-Layer – Default Economic und Resilience Policy – steuern die Regelwerke zentral.
