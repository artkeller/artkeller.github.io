# Operating Modes

The software provides different operating modes that adjust the user interface and behavior depending on device type and usage preferences.

All modes share the same underlying codebase and functionality.

## Modes

* **base**
  Simplified interface optimized for mobile devices and basic usage.

* **smart**
  Extended dashboard-style interface for larger devices (e.g. tablets, notebooks),
  potentially with data-related limitations.

* **panel**
  Interface optimized for large displays (e.g. wide-screen monitors),
  including persistent views and fewer built-in limitations.

These modes are **configuration variants only** and do not represent different product tiers or service levels.

The presence of extended functionality or reduced limitations in certain modes does not imply fitness for professional or production use.
